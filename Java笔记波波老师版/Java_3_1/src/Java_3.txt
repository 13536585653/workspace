1.类与对象
  类是抽象的，泛指某一类具有相同属性或方法的一类事物。
  对象是具体的，是某个类的一个具体的实例。 
  
 java编程里，先编写类(class),再通过类去生成对象。
 
 在分析问题时，先有对象，再总结抽象出类。

 
 2.类的写法:
   java里的最基本的单元。就是类(class) ，类里包含属性和方法。
   属性可以理解为名词，方法可以理解为 动词 

   class 类名{
   
   
        属性1;
        属性2;
         
       方法1;
       方法2;
   
   }
   
   注意：类名 首字母大写
   
 3. 通过类去实例化对象
 
      类名   对象名=new 类名(); 
      
 4. 访问对象的属性，和方法
           对象.属性
           对象.方法()
 
 
 5.类里 toString方法的作用
      类实际上是一种复杂的数据类型
      int x=5;
      Dog dog=new Dog(); 
      输出一个对象时 ,  
       如果一个类 ，没有toString方法
       则 输出的是该对象的类型和在内存中的地址 
         如 Dog@12d96f2
      如果有toString方法 ，
      则输出toString方法里的返回的内容 
      
      toString 方法必须:
      public String toString(){
         return "xxxxx";
      }
      
      toString方法里，通常输出该类的全部属性。
